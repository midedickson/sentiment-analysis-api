To Install

1. Create Virtual Environment

2. run:
   `pip install -r requirements.txt`

3. Open python shell
   run:

   `>>> import nltk`

   `>>> nltk.download("puntk_tab")`

   `>>> nltk.download("stopwords")`

4. Add required CSV files to the root directory

5. run:
   `sudo fastapi dev main.py`
